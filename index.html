Instructions


1. You must currently run this in IE.
2. On loading you must allow ActiveX controls. (Allow blocked content)
3. You may run some basic test functions that are listed in the Options drop down.
4. The test functions are designed to be used on the MAXIS/SELF Menu.
5. If you are getting connection errors you must check your session ID and verify that it's the same as
    the ID in Bluezone.    ( Options -> API... -> Short Name Session Identifier < ex. a, b, c, d... > )


Note: To change the session ID navigate to sitescripts/FUNCTIONS.js and go to the function bzConnect(). Change the session ID to match your open session ID.
